# **Aeon's Archive - Project Site Map**


_This document serves as a guide for organizing the exported HTML files for Aeon's Archive_.

## I. Public Library (Guest Access)

- index.html - Home (Desktop/Mobile): The landing page for the archive.

- archive.html - The Eternal Library: The full collection of poems, stories, and novels.

- chronicle.html - Timeline Archive: The chronological record of the archive's growth.

- reading-suite.html -Reading Experience: The immersive, parchment-styled reader for manuscripts.


## II. Curator's Suite (Private Management)

- login.html - Curator Login: Secure entry point for the archive owner.

- otp-verify.html - Verification: Two-step authentication screen.

- dashboard.html - Master Ledger: The central management hub to view, edit, or delete works.

- submit.html - New Manuscript: The interface for adding new works to the archive.


## III. Hosting Instructions (GitHub Pages)

To launch Aeon's Archive as a live website on GitHub Pages:

### 1. Create a GitHub Account: 

- If you don't have one, sign up at github.com.

### 2. Create a Repository:

- Click the "+" icon in the top right and select New repository.

- Name it aeons-archive.

- Set it to Public and check Add a README file.

- Click Create repository.

### 3. Upload Your Code:

- In your new repository, click Add file > Create new file.

- Name the first file index.html.

- Paste the code provided by Stitch for the Home screen.

- Click Commit changes.

- Repeat this for all other screens (e.g., archive.html, login.html).

### 4. Activate GitHub Pages:

- Go to Settings > Pages (in the left sidebar).

- Under Build and deployment, ensure the Source is set to Deploy from a branch.

- Under Branch, select main and / (root), then click Save.

### 5. View Your Site:

- Wait a minute or two. GitHub will provide a link (usually https://username.github.io/aeons-archive/). Your archive is now live!
